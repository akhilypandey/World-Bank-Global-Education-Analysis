# 🌍 World Bank Global Education Analysis

## 📌 Project Overview
This project analyzes global education indicators using the World Bank EdStats dataset to uncover patterns in literacy rates across countries, regions, and income groups.

The objective is to transform raw data into meaningful insights that can help policymakers, governments, and global organizations improve education systems and reduce inequality.

---

## 🎯 Problem Statement
Analyze global literacy trends and identify key factors influencing education outcomes across countries.

---

## 🎯 Business Objective
- Identify low-performing regions and countries
- Understand impact of economic conditions on literacy
- Provide data-driven recommendations for improving education systems

---

## 📊 Dataset
The project uses 5 datasets from World Bank EdStats:
- EdStatsData.csv (main data)
- EdStatsCountry.csv (country-level info)
- EdStatsSeries.csv
- EdStatsCountry-Series.csv
- EdStatsFootNote.csv

---

## 🧹 Data Processing
- Filtered literacy-related indicators
- Converted wide data to long format
- Handled missing values
- Merged country dataset (Region + Income Group)

---

## 📈 Exploratory Data Analysis (EDA)
- Performed Univariate, Bivariate, and Multivariate Analysis
- Created 20+ visualizations
- Identified patterns in:
  - Literacy distribution
  - Regional inequality
  - Income group differences
  - Time-based trends

---

## 🔍 Key Insights
- Literacy rates have improved globally over time
- High-income countries consistently outperform others
- Strong inequality exists across regions
- Economic condition is a major driver of literacy

---

## 💡 Business Recommendations
- Focus on low-income regions
- Implement region-specific policies
- Improve efficiency of education spending
- Use data-driven decision making

---

## 🧠 Tools & Technologies

- Jupyter Notebook / Google Colab

---

## 🚀 Project Outcome
This project provides actionable insights that can help improve global education systems and reduce inequality.
